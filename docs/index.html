What it does

Built to learn RAG concepts — ingest official docs, retrieve with hybrid search, generate cited answers, measure with RAGAS.

🔍 Hybrid retrieval

Semantic search (Qdrant) + keyword search (BM25) merged with RRF, then reranked with a cross-encoder.

💬 Grounded answers

Local LLM (Ollama) answers only from retrieved doc chunks — with title, section, and URL citations.

📊 Measured quality

RAGAS evaluation on 38 questions. Ablation study proves hybrid beats vector-only retrieval.

Sample question
Question
How do I create a Kubernetes Deployment?
Answer
To create a Deployment in Kubernetes, you define it in a YAML manifest and apply it with kubectl. Example: kubectl apply -f https://k8s.io/examples/application/deployment.yaml A Deployment manages ReplicaSets and ensures the desired number of Pod replicas are running. You can check status with: kubectl get deployments The Deployment controller handles rolling updates and rollbacks automatically.
